# ✨ WeaponEnchanter - Minecraft Plugin

![Minecraft](https://img.shields.io/badge/Minecraft-1.21.8-brightgreen)
![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![Java](https://img.shields.io/badge/Java-21-orange)

> **Add custom effects to ANY weapon!** This plugin lets you enchant any weapon with special effects like fire, poison, lightning, and more!

---

## ⚔️ Available Effects

### 🔥 Fire Effect
- **Command:** `/we fire`
- **Effect:** Sets enemies on fire for 5 seconds
- **Particles:** Flame particles
- **Perfect for:** Continuous damage over time

### ☠️ Poison Effect
- **Command:** `/we poison`
- **Effect:** Applies Poison III for 5 seconds
- **Particles:** Green poison particles
- **Perfect for:** Weakening tough enemies

### ⚡ Lightning Effect
- **Command:** `/we lightning`
- **Effect:** Strikes lightning on hit + 4 bonus damage
- **Particles:** Electric sparks
- **Perfect for:** Massive burst damage

### 💀 Wither Effect
- **Command:** `/we wither`
- **Effect:** Applies Wither III for 4 seconds
- **Particles:** Soul particles
- **Perfect for:** Draining enemy health

### 👁️ Blindness Effect
- **Command:** `/we blind`
- **Effect:** Blindness + Darkness for 5 seconds
- **Particles:** Smoke particles
- **Perfect for:** Disorienting enemies

### ❄️ Slowness Effect
- **Command:** `/we slow`
- **Effect:** Slowness V + Mining Fatigue III for 6 seconds
- **Particles:** Snowflake particles
- **Perfect for:** Crowd control

### ♥️ Life Steal Effect
- **Command:** `/we lifesteal`
- **Effect:** Heals 40% of damage dealt
- **Particles:** Heart particles
- **Perfect for:** Sustain in combat

### 💥 Explosion Effect
- **Command:** `/we explode`
- **Effect:** Creates explosion on hit
- **Particles:** Explosion particles
- **Perfect for:** Area damage

### 🧊 Freeze Effect
- **Command:** `/we freeze`
- **Effect:** Extreme slowness + freeze status
- **Particles:** Snowflake + ice particles
- **Perfect for:** Completely stopping enemies

### 💨 Mega Knockback
- **Command:** `/we knockback`
- **Effect:** Launches enemies far away
- **Particles:** Cloud particles
- **Perfect for:** Keeping distance

---

## 🎮 How to Use

1. **Hold any weapon** (sword, axe, bow, trident, etc.)
2. **Type the command** `/we <effect>` (e.g., `/we fire`)
3. **Hit enemies** to trigger the effect!
4. **Stack multiple effects** on the same weapon!

### Example:
```
/we fire
/we poison
/we lifesteal
```
Now your weapon has Fire + Poison + Life Steal! 🔥☠️♥️

---

## 📦 Installation

1. **Download** the latest `WeaponEnchanter.jar` from releases
2. **Place** the jar file in your server's `plugins/` folder
3. **Restart** your server
4. **Enjoy** enchanting weapons!

---

## 🛠️ Building from Source

### Prerequisites
- Java 21 or higher
- Maven

### Build Steps
```bash
cd WeaponEnchanter
mvn clean package
```

The compiled jar will be in `target/WeaponEnchanter.jar`
